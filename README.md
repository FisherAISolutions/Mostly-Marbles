# Mostly Marbles

Mostly Marbles is a marble marketplace and AI-assisted identifier.

This starter includes:

- Next.js App Router
- Supabase client wiring (frontend only)
- Pages:
  - Home
  - Marketplace
  - AI Identifier
  - Admin Dashboard
  - Admin Roles
  - Admin Training
- Simple role-based guard component (stubbed)
- Dark professional theme

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.local.example` to `.env.local` and fill in values.

3. Run dev server:

   ```bash
   npm run dev
   ```

Then open http://localhost:3000.
